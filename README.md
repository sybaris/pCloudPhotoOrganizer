# pCloudPhotoOrganizer

Application Android en .NET MAUI permettant :

- d'afficher les photos du téléphone avec miniatures (style Galerie)
- de sélectionner plusieurs photos
- de grouper les photos par date
- de créer automatiquement un dossier pCloud nommé "YYYY.MM.DD - "
- d’uploader photos et vidéos vers un dossier cible pCloud
- de configurer les dossiers source locaux et le dossier pCloud cible
- de stocker un token pCloud de manière sécurisée

## Technologies
- .NET 8
- .NET MAUI
- MediaStore Android
- pCloud REST API
- MVVM

## Compilation
Ouvrir le projet avec Visual Studio 2022 + workload MAUI + SDK Android.
